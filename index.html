Aerfaying-Explore

提供优化、补丁及小功能提升社区内的探索效率和用户体验

这是什么

Aerfaying-Explore 是一个开源的 Userscript 脚本，为阿儿法营 / 稽木世界社区提供了一些方便的小功能（如「显示评论详细信息」等）来优化社区的探索体验。

欢迎在 Github 仓库 waterblock79/aerfaying-explore 中通过 Issue 功能提交问题或审查源代码！

对于 iOS 用户：
1. 打开 设置 - Safari 浏览器 - 扩展 - Userscripts - 开启
2. 打开 Userscripts 应用，点击 Set Userscripts Directory 摁钮，选择一个方便找到的文件夹。
3.A. 在电脑上打开该链接（备用地址）并 Ctrl+S 保存该文件，通过 QQ 等方式传输到手机上，然后将这个文件转存在你刚刚选择的文件夹中即可。
3.B. 如果无法使用电脑，请下载该压缩包，解压压缩包并将其中的 aerfaying-explore.user.js 文件转存在你刚刚选择的文件夹中。
4. 打开 Safari，点击地址栏右侧拼图图标，点击 Userscripts，出现访问的权限请求时点击 始终允许。
5. 弹出 Safari 中的 Userscripts 插件设置框则代表配置成功，现在即可使用脚本了！
